Project Title: Graffiti Art Portfolio Website
Description:
The Graffiti Art Portfolio Website is a creative and visually appealing web application designed to showcase graffiti artwork. It features a modern layout, intuitive navigation, and a contact form for potential collaborators. This project demonstrates responsive design principles and provides an engaging user experience.
Features: 1. Header with Navigation Bar: • Logo and menu links to Home, Portfolio, Blog, and Contact sections. • Smooth and structured layout. 2. Main Section: • A bold heading: “Unleash Your Creativity.” • A grid-based image gallery showcasing different graffiti themes: • Urban Jungle • City Vibe • Graffiti Dreams • Street Art • Urban Legends • Concrete Jungle • City Lights • Graffiti Vibe • Urban Dreams • Images are categorized into rows for easy navigation. 3. Contact Section: • Form with placeholders to input data. • Skip and Submit buttons for user interaction. • Clear call-to-action: “If you are interested in working together, please fill out the form below.” 4. Testimonials: • Section titled “What People Say.” • Feedback from art enthusiasts, collectors, and gallery owners. • Each testimonial includes a profile image, name, role, and comment. 5. Footer: • Copyright information. • A Contact button for easy accessibility.
Project Structure:
Files: • index.html: The main HTML file containing the structure of the web page. • stylesheet.css: The CSS file for styling and layout. • Assets/: Folder containing all the images used in the project.
Folders: • Assets/: • Contains images for gallery items, profile pictures, and other design elements.
Technologies Used: 1. HTML5: For structuring the content of the website. 2. CSS3: For styling and layout. 3. Responsive Design: Optimized for various devices.
Setup Instructions: 1. Clone the repository or download the project files. 2. Ensure all files are placed in the same directory, maintaining the folder structure. 3. Open the index.html file in a web browser to view the project.
Future Enhancements: 1. Add interactivity using JavaScript. 2. Improve form validation and submission functionality. 3. Optimize images for faster loading. 4. Make the navigation bar sticky for better usability.
