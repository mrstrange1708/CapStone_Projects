Capstone Project: Graffiti Art Showcase 🎨

Project Overview

This Capstone Project is a visually appealing and responsive Graffiti Art Showcase Website designed to display artwork, connect with potential collaborators, and feature testimonials from art enthusiasts. The website emphasizes modern design principles, smooth navigation, and adaptability across devices.

Features
	•	🎨 Beautiful Home Page: Eye-catching hero section with a focus on unleashing creativity.
	•	🖼️ Image Gallery: A collection of graffiti artworks displayed in a responsive and grid-based layout.
	•	📝 Contact Form: Users can reach out by filling out a streamlined contact form.
	•	🌟 Testimonials Section: A horizontally scrollable section featuring feedback from art collectors, gallery owners, and enthusiasts.
	•	📱 Fully Responsive: The website adapts to different screen sizes, from desktops to tablets and mobile devices.

Technologies Used
	•	HTML5: Structure of the website.
	•	CSS3: Custom styling and responsive design using flexbox and media queries.
